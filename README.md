# 🌍 AfriConnect Summit 2026

Bienvenue sur **AfriConnect Summit 2026**, un site web événementiel moderne conçu pour promouvoir un sommet dédié à l'innovation, à la technologie et à l'entrepreneuriat en Afrique.

Le projet a été réalisé entièrement avec **HTML5**, **CSS3** et **JavaScript**, sans framework, dans le but de mettre en pratique les compétences en développement web front-end.

---

## 📌 Présentation

AfriConnect Summit est une conférence fictive qui rassemble :

- 🚀 Entrepreneurs
- 💻 Développeurs
- 🤖 Experts en Intelligence Artificielle
- 📊 Data Scientists
- 🎨 Designers UX/UI
- 🌍 Investisseurs et acteurs du numérique

Le site permet aux visiteurs de découvrir l'événement, consulter le programme, les intervenants et s'inscrire.

---

## ✨ Fonctionnalités

- 🏠 Page d'accueil moderne
- 📅 Programme interactif sur 3 jours
- 👨‍💼 Présentation des intervenants
- 📞 Formulaire d'inscription avec validation JavaScript
- 🌙 Mode sombre (Dark Mode)
- ⏳ Compteur dynamique avant l'événement
- 🔍 Navigation responsive
- 📱 Compatible mobiles, tablettes et ordinateurs
- ⬆️ Bouton "Retour en haut"
- 🎬 Vidéo de présentation
- 🎨 Animations CSS et JavaScript
